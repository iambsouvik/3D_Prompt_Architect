# 🎨 3D Asset Prompt Architect Engines

A premium, production-ready AI Prompt Generator engineered specifically for 3D artists, digital creators, and designers. This full-stack local engine takes raw conceptual descriptions and dynamically transforms them into master-level, high-fidelity prompts optimized for top-tier generative AI models.

---

## 🚀 Key Features

- **Multi-Model Optimization:** Tailor-made algorithmic tuning for **Midjourney v6**, **Stable Diffusion XL (SDXL)**, and **DALL-E 3**.
- **Dynamic Structural Controls:** Fine-tune deep parameters including **3D Base Styles** (Pixar, PVC Figurine, Caricature, Blender Cycles Render), **Lighting Setups** (Rim light, Volumetric Neon, Chiaroscuro), and complex **Material Properties** (Subsurface Scattering, Matte Vinyl, Frosted Glass).
- **Pro-Layout Engine:** Multi-tab layout interface generating parallel asset outputs (**Cinematic View** vs. **Dynamic Composition**) alongside a comprehensive automated **Negative Prompt** array.
- **Cyberpunk Dark Aesthetic:** A polished, modern visual dashboard designed for comfortable, prolonged creative workflows.

---

## 🛠️ Technology Stack

- **Frontend/UI:** Streamlit (Custom Premium CSS Integration)
- **Backend Infrastructure:** Python 3.x & HTTP API Endpoint Architecture
- **Core AI Model:** Google Gemini 2.5 Flash
